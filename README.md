# floor manager discord bot

[![discord](https://img.shields.io/discord/1328881859597307987?color=7289da&logo=discord&logoColor=white)](https://discord.gg/9Fpt5wm84V)
[![version](https://img.shields.io/badge/version-0.0.1-blue.svg)](https://github.com/wwidjaja0/floor-manager)
[![license](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

a discord bot for managing the "floor" for the poker club @ uc san diego discord server

## features

- table management
- ledgers
- p2p payment verification
- host accountability
- edu email verification

see the [design document](doc/design.md) for full design details

## running locally, contributing, issues, and support

we welcome contributions! to run locally, see [contributing guidelines](CONTRIBUTING.md) for details

for bug reports, feature requests, etc, see [github issues](https://github.com/wwidjaja0/floor-manager/issues)

for support, join the [poker club @ uc san diego discord server](https://discord.gg/9Fpt5wm84V).
